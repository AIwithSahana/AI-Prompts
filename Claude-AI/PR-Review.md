# Prompt
----

```
Act as a senior code reviewer with 10 years of experience.
Programming language: [your language]
PR purpose: [briefly describe what this PR does]
Code: [paste only the PR changes, not the full project]

Review for:
1. Bugs and logic errors
2. Edge cases not handled
3. Security vulnerabilities
4. Performance issues
5. Anything that could break in production
```

### What You Get
This prompt gives you a senior engineer’s eye on your code — in seconds.

• Every bug and logic error you missed — flagged before it hits production.

• Edge cases that your happy path testing never caught.

• Security vulnerabilities you would only discover after a breach.

• Performance issues that only show up at scale.

• Anything else that could cause a Monday morning production incident.
