## PROMPT
----------------------

```
I’m a [Role – Backend Dev / Frontend Dev / Full Stack].
I’ve been assigned to implement the following feature in our production system:
Feature Name: [Name]
Business Goal: [Why this feature exists]
User Flow: [Step-by-step expected behavior]
Current Tech Stack:
•	Backend: [e.g., Node.js + Express]
•	Frontend: [e.g., React]
•	Database: [e.g., PostgreSQL]
•	Hosting: [e.g., AWS]
Existing Architecture Constraints:
•	[Monolith / Microservices]
•	Auth mechanism: [JWT/OAuth/etc.]
•	API style: [REST/GraphQL]
Please help me with:
1.	Breaking this into clear technical tasks (backend, frontend, DB).
2.	Designing database schema changes (if required).
3.	API contract design (request/response examples).
4.	Edge cases and failure scenarios.
5.	Security considerations.
6.	Testing strategy (unit, integration).
7.	Performance considerations at scale.
Provide a production-ready structured plan, not just high-level advice.
```
