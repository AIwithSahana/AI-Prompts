# Developer Productivity Prompts

## Prompt #1 — Bug Root Cause Analysis

### The Prompt

```

Find the root cause of this bug. Explain why it happens. Suggest the cleanest fix.

Code: [paste your code here]

```

### What You Get

- The exact **root cause of the bug**
- A **clear explanation** of why it’s happening
- The **cleanest possible fix** you can implement immediately

# Prompt #2 — Documentation Writing

Let’s be honest — nobody enjoys writing documentation.

But organizations care about it deeply. And they notice the engineer who writes clear, thorough documentation. Use this prompt to generate documentation quickly and properly.

### The Prompt

```

Write clear technical documentation for this code.

Include purpose, inputs, outputs, and edge cases.

Code: [paste your code here]

```

### What You Get

- A **clear one-line explanation** of what the code does
- Every **input and output properly documented**
- **Edge cases clearly listed** with nothing left ambiguous

# Prompt #3 — SQL Query Generation

SQL feels easy — until performance actually matters.

When you have millions of rows in production and a slow query starts causing issues, that’s when unoptimized SQL becomes expensive. This prompt helps generate optimized queries from the start.

### The Prompt

```

Generate an optimized SQL query for: [describe your requirement]

Explain indexes or performance considerations.

```

### What You Get

- A **production-ready optimized SQL query**
- Recommended **indexes to use — and why**
- **Performance considerations** before deploying to production
